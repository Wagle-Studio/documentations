# Documentations

⚡ Next.js · ⚛️ React · 🎨 SCSS · 📝 MDX

## Présentation

Site de documentation personnel regroupant des cours et références techniques rédigés en MDX.

ℹ️ Initialement conçue pour mon usage personnel, j'ai ensuite partagé cette documentation dans le cadre de mes activités de formateur en conception et développement web.

⚠️ Je procède actuellement à la migration du contenu depuis Notion. J'en profite pour retravailler le fond et la forme, ce travail est donc encore en cours.

---

## Installation

```sh
git clone git@github.com:Wagle-Studio/documentations.git
cd documentations
npm install
npm run dev
```
